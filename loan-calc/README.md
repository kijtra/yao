# loan-calc

ローン計算ツールサイト。詳細仕様は親ディレクトリの `PROJECT.md` を参照。

## 開発コマンド

```sh
pnpm install         # 依存インストール
pnpm dev             # 開発サーバ起動（http://localhost:4321）
pnpm build           # 本番ビルド（dist/）
pnpm preview         # ビルド結果のプレビュー
```

## スタック

- Astro 6 (minimal template)
- TypeScript strict
- Tailwind CSS v4
- pnpm
